Cross-platform desktop chat application, completely open source

A special thanks to all of the amazing resources that made this project possible, specially to the Electron platform, the beautifully designed David's Chat UI and all of the other tons of dependencies backed by NPM and Bower repositories.

One-click free deployment at Heroku

Start right now by using the Heroku Button below. It will automatically create all the required resources for serving your own chat room, completely free!

Why?

Working on a undergrad project for Distributed Systems Course at IC/UNICAMP ministrated by Prof. Leandro Villas, I realized that, although a lot of cool, clean and smart technologies were openly available, it was really hard to find a subset that could be customized directly to work almost out of the box for simple messaging needs.
